项目文件清单

1. audio.m4a
   当前示例音频，已替换为《班级小故事》，并优化为更适合浏览器流式播放的版本。

2. index.html
   微信扫码后的音频播放页，支持播放、暂停、快进快退、倍速、进度拖动和下载，当前主题色已同步到更高级的色系。

3. qrcode-generator.html
   根据 GitHub Pages 播放页规则生成二维码海报，支持主题、标题、海报副标题、播放页副标题和角标配置，当前海报主题已调整为更沉稳的高级感风格。

4. README.md
   当前版本总说明、功能清单和使用方法。

5. PRD-v1.0.md
   v1.0 历史 PRD 文档，用于保留早期需求记录。

6. PRD-v1.1.md
   v1.1 历史 PRD 文档。

7. PRD-v1.2.md
   与当前代码实现保持一致的 PRD 文档。

8. GitHub-Token-创建说明.md
   GitHub Token 的创建与权限说明。

9. GitHub-Pages-Build-and-Deployment-说明.md
   GitHub Pages 的发布配置说明。

10. 🚀快速开始-三分钟完成.md
    中文快速部署指南。

    Windows 一键打开二维码生成器。

12. deploy.sh
    macOS/Linux 一键打开二维码生成器。

当前版本功能

- 微信端与手机端优先的单页音频播放页
- 7 套主题样式，二维码海报与播放页主题同步
- 播放、暂停、快进、快退、倍速、进度拖动、下载音频
- 海报副标题与播放页副标题可分别自定义
- 副标题中的 `-` 或换行可拆分为多行展示
- 二维码海报生成与 PNG 导出
- 生成器按钮状态提示、脚本校验与下载稳定性增强
- 默认使用 GitHub Pages 同域音频地址
- 外部音频加载失败时，播放器可回退到站内默认音频
- 海报页与播放页使用更统一的高级感主题色
- `故事朗读` 和 `自然清听` 主题已针对班级故事场景细调

当前版本已移除

- 播放页复制链接按钮
- 播放页底部说明文案

使用提醒

- 仓库必须为 Public，否则 GitHub Pages 分享无法正常工作。
- 建议保持音频文件名为 audio.m4a，便于二维码长期复用。
- 生成器中的分支名字段会保留显示和预填，但默认 GitHub Pages 音频路径并不依赖该字段。
- 二维码默认指向 GitHub Pages 播放页，而不是直接跳转音频文件。
