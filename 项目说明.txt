项目文件清单

1. audio.m4a
   本地示例音频文件，可直接上传到 GitHub。

2. index.html
   微信扫码后的音频播放页，支持播放、暂停、快进快退、倍速和下载。

3. qrcode-generator.html
   根据 GitHub 用户名、仓库名和音频主题生成美化二维码海报。

4. README.md
   项目总说明和功能清单。

5. PRD-v1.0.md
   与当前代码一致的产品需求说明。

6. GitHub-Token-创建说明.md
   GitHub 临时 Token 的创建与权限说明。

7. GitHub-Pages-Build-and-Deployment-说明.md
   GitHub Pages 的 Build and deployment 配置说明。

8. 🚀快速开始-三分钟完成.md
   中文快速部署指南。

9. 立即开始-三步完成.bat
   Windows 一键打开二维码生成器。

10. deploy.sh
   macOS/Linux 一键打开二维码生成器。

当前版本功能

- 微信端手机音频播放页
- 7 套主题样式
- 播放、暂停、快进、快退、倍速、进度拖动
- 复制页面链接、下载音频
- 二维码海报生成与 PNG 导出
- GitHub Pages + jsDelivr 免费部署

使用提醒

- 仓库必须是 Public，否则 GitHub Pages 和 CDN 外链无法正常分享。
- 建议保持音频文件名为 audio.m4a，便于二维码长期复用。
- 二维码实际指向 GitHub Pages 播放页，而不是直接指向音频文件，因此体验更适合微信打开。
