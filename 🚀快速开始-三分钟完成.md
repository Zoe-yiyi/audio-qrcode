# 微信音频二维码快速开始

## 第 1 步：上传到 GitHub

1. 打开 `https://github.com/new`
2. 创建公开仓库：`audio-qrcode`
3. 上传这些文件到仓库根目录：
   - `audio.m4a`
   - `index.html`
   - `qrcode-generator.html`
4. 进入 GitHub 仓库设置 `Settings -> Pages`
5. 在 `Build and deployment` 中选择：
   - `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`

## 第 2 步：确认音频链接

推荐使用 jsDelivr 加速链接：

```text
https://cdn.jsdelivr.net/gh/你的用户名/audio-qrcode@main/audio.m4a
```

如果你已经拿到原始链接，也可以直接填进二维码生成器。

## 第 3 步：生成并下载二维码海报

1. 用浏览器打开本地文件 `qrcode-generator.html`
2. 输入 GitHub 用户名、仓库名
3. 选择适合音频气质的主题
4. 可选填写标题、副标题和角标文案
5. 点击 `生成二维码`
6. 点击 `下载二维码海报`

## 建议主题

- 冥想、钢琴、轻音乐：`空灵氛围`
- 访谈、课程、播客：`播客对谈`
- 故事、朗读、童声：`故事朗读`
- 备忘、提醒、短语音：`语音备忘`
- 品牌宣传、活动物料：`金色奢感`

## 当前版本已支持

- 微信手机端播放页
- 7 套主题风格
- 播放/暂停、快进快退、进度拖动、倍速
- 复制链接、下载音频
- PNG 二维码海报导出

## 微信使用建议

- 把下载后的二维码海报发到微信聊天、朋友圈或公众号素材库
- 也可以直接打印到名片、海报、宣传页
- 如果后续更换音频，建议保持文件名 `audio.m4a` 不变，这样旧二维码仍可复用
